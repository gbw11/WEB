# HTML Emmet

## 1. Emmet(에밋)이란?

Emmet은 HTML, XML, CSS 등의 코드를 기계적으로 타이핑하는 시간을 획기적으로 줄여주는 **동적 코드 생성 엔진**입니다. 과거에는 'Zen Coding(젠 코딩)'이라고 불렸으며, 현재는 VSCode를 비롯한 대부분의 코드 에디터에 기본적으로 내장되어 있습니다.

> Emmet Cheat sheet
>
> - https://docs.emmet.io/cheat-sheet/



### 1.1 Emmet의 핵심 작동 원리
Emmet의 가장 중요한 철학은 **CSS 선택자(Selector) 문법을 그대로 차용**했다는 점입니다. 
보통 CSS는 이미 만들어진 HTML 요소를 '선택'해서 꾸밀 때 사용하지만, Emmet은 이 문법을 역으로 활용하여 **HTML 구조를 '생성'**하는 데 사용합니다.

* **작동 프로세스:** `축약어 입력` ➡️ `Tab 또는 Enter 키 누름` ➡️ `완성된 HTML 코드로 자동 변환`

##  2. Emmet 핵심 문법 및 실전 예시

### 2.1 HTML 문서 뼈대
가장 기본이 되는 HTML5 문서를 완성합니다.

| Emmet 명령어 | 설명                 | 생성되는 코드                    |
| :----------- | :------------------- | :------------------------------- |
| `!`          | HTML5 기본 뼈대 생성 | `<!DOCTYPE html>`<br>`<html>`... |



### 2.2 요소 배치하기: 구조 잡기

태그를 직접 열고 닫을 필요 없이 문서의 뼈대를 바로 설계합니다.

| Emmet 명령어 | 기호 | 설명                                                 | 생성되는 코드              |
| :----------- | :--- | :--------------------------------------------------- | :------------------------- |
| `ul>li`      | `>`  | **자식 요소:** 태그 안(하위)으로 들어갑니다.         | `<ul><li></li></ul>`       |
| `h1+p`       | `+`  | **형제 요소:** 태그를 같은 레벨에 나란히 배치합니다. | `<h1></h1>`<br>`<p></p>`   |
| `div^p`      | `^`  | **위로 올라가기:** 부모 레벨로 빠져나옵니다.         | `<div></div>`<br>`<p></p>` |



### 2.3 ID와 Class (CSS 선택자 문법 적용)

CSS를 적용하기 위한 선택자를 태그 생성과 동시에 부여합니다. 

> 태그를 생략하고 `.box`만 적으면 자동으로 `div`가 생성됩니다.

| Emmet 명령어     | 설명                                 | 생성되는 코드                          |
| :--------------- | :----------------------------------- | :------------------------------------- |
| `div#header`     | `header`라는 ID를 가진 div 생성      | `<div id="header"></div>`              |
| `p.text-bold`    | `text-bold`라는 클래스를 가진 p 생성 | `<p class="text-bold"></p>`            |
| `form#login.box` | ID와 클래스를 동시에 부여            | `<form id="login" class="box"></form>` |



### 2.4 반복과 넘버링

목록이나 게시물 카드를 여러 개 만들 때 매우 유용합니다.

| Emmet 명령어 | 기호 | 설명                                                | 생성되는 코드                                                |
| :----------- | :--- | :-------------------------------------------------- | :----------------------------------------------------------- |
| `li*3`       | `*`  | **반복:** 요소를 지정한 숫자만큼 생성합니다.        | `<li></li>` (3개)                                            |
| `li.item$*3` | `$`  | **넘버링:** 달러 기호 자리에 숫자가 1부터 붙습니다. | `<li class="item1"></li>`<br>`<li class="item2"></li>`<br>`<li class="item3"></li>` |



### 2.5 디테일 추가: 텍스트와 속성

태그 사이의 글자나 이미지 주소 등을 함께 작성합니다.

| Emmet 명령어   | 기호 | 설명                                              | 생성되는 코드           |
| :------------- | :--- | :------------------------------------------------ | :---------------------- |
| `button{클릭}` | `{}` | **텍스트:** 태그 내부에 들어갈 텍스트를 적습니다. | `<button>클릭</button>` |
| `a[href="#"]`  | `[]` | **속성:** 태그의 속성을 지정합니다.               | `<a href="#"></a>`      |

## 3. 응용: 내비게이션 바 만들기

**입력해 보세요**

```html
nav>ul.menu>li.nav-item*4>a{메뉴 $}
```



**결과**

```html
<nav>
  <ul class="menu">
    <li class="nav-item"><a href="">메뉴 1</a></li>
    <li class="nav-item"><a href="">메뉴 2</a></li>
    <li class="nav-item"><a href="">메뉴 3</a></li>
    <li class="nav-item"><a href="">메뉴 4</a></li>
  </ul>
</nav>
